# NovaTech — Responsiv Landing Page
Rəqəmsal həllər studiyası üçün hazırlanmış, tam responsiv landing page.

## Layihənin qısa təsviri
Semantik HTML5 strukturu üzərində qurulmuş, mobile-first yanaşma ilə 
3 breakpoint-də (mobil/tablet/desktop) responsiv dizayn edilmiş sayt. 
6 bölmədən ibarətdir: Hero, Haqqımızda, Xidmətlər, Nümunə Qalereya, 
Əlaqə formu və Footer.

## İstifadə olunan texnologiyalar
- HTML5 (semantik struktur)
- CSS3 (Flexbox, Grid, media query-lər)
- Vanilla JavaScript (DOM manipulyasiyası, event listener-lər)

## Canlı demo
https://vusalamirzazada.github.io/NOVATECH-Website/

## Xüsusiyyətlər
- Mobil menyu toggle (hamburger düymə)
- Naviqasiyada smooth scroll
- Qalereya kartlarında klikləmə ilə ətraflı məlumat
- Əlaqə formunda client-side validasiya (boş sahə, email regex)
- Əlçatanlıq: alt-text, ARIA atributları, klaviatura naviqasiyası
